# roku-remote

Simple command-line based remote for the Roku 3.

## Usage

```
$> ./remote.py <ip-addr>
```

### Commands:

Note: No output is currently displayed.

Up - Keypad Up
Down - Keypad Down
Left - Keypad Left
Right - Keypad Right
Select - Numpad Enter (currently only supports numpad enter key)
Back - Delete
Home - Home
Quit - q
